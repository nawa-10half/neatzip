どこでも“ゴミなし”で開ける、クリーンな ZIP を作るアプリです。

macOS の Finder で圧縮すると、__MACOSX・.DS_Store・._* といった Mac 固有の隠しファイルが一緒に入り、Windows や Linux では「ゴミ」として見えてしまいます。NeatZip は ZIP をゼロから組み立て、あなたが選んだファイルだけを追加します。だから「後から掃除する」のではなく、最初からクリーンです。

最初からクリーン
__MACOSX・.DS_Store・._*（AppleDouble）を、そもそも書き込みません。ZIP の中身は Finder で見えるファイルそのまま。余計なものは一切入りません。

右クリック / ドラッグ&ドロップ
Finder でファイルやフォルダを右クリックして「Clean ZIP with NeatZip…」を選ぶか、ウインドウにドロップするだけ。ワンステップで完了します。

任意のパスワード
必要なときはパスワードを設定できます。互換性重視の ZipCrypto（Windows でダブルクリックで開ける）か、強力な AES-256 を選べます。

高速
並列圧縮により、標準の Finder 圧縮よりも体感で速く、どこでも開ける標準的な ZIP を生成します。

圧縮専用（解凍機能はありません）
NeatZip は ZIP を「作る」ためのアプリです。ZIP の解凍（展開）機能はありません。圧縮に特化したツールであり、展開ツールではありません（macOS なら ZIP はダブルクリックで展開できます）。

単機能・プライバシー重視
NeatZip は、ひとつのことだけをきちんとやります。アカウント不要・ログイン不要・データ収集なし。完全オフラインで動作し、ファイルが Mac の外に出ることはありません。
